AI Logistics Control Tower 🚚

A full-stack AI-powered logistics dashboard to monitor delivery KPIs and improve operational efficiency.

Features
Submit logistics data like orders, delays, and costs
View all submitted data in real-time
Analyze delivery KPIs with interactive charts
Predict delays using operational data
Technologies
Python
Flask for backend API
Streamlit for frontend dashboard
SQLite for database
Pandas, scikit-learn, and requests for data handling
Folder Structure

ai-logistics-control-tower/
├── backend/
│ └── api.py (Flask backend API)
├── frontend/
│ └── app.py (Streamlit frontend dashboard)
├── data/ (datasets or scripts)
└── requirements.txt (Python dependencies)
